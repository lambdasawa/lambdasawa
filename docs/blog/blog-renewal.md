今までは www.lambdasawa.net に Hugo で作ったサイトを CloudFront  + S3 の構成で公開していました。
新しく www.lambdasawa.me のドメインを取得して Next.js + Tailwind CSS + microCMS でこのサイトを作りました。

旧ドメインには Lambda@Edge が仕掛けてあって、今はアクセスすると新ドメインにリダイレクトされるようになっています。

仕事では基本的にはバックエンドエンジニアとして振る舞うことが多いのですが、最近はフロントエンドエンジニア的な振る舞いをする機会が生まれてきたので、経験値を積むために Next.js を採用しました。
microCMS も同様に仕事で使う機会が増えてきたことが採用理由です。

JavaScript (TypeScript) 自体は今までも頻繁に書いていたし、 microCMS も良い意味でよくある Web API なので特に躓くことは無く使えています。
CSS は未だに上手に書けないので、このサイトの構築/運用で克服していきたいです。

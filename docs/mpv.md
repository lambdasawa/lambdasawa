# mpv

- <https://github.com/mpv-player/mpv>
- <https://mpv.io/manual/master/#keyboard-control>

## YouTube 1曲リピート

```sh
mpv --loop --no-video --volume=50 'https://www.youtube.com/watch?v=xxxx'
```

## YouTube ストリーミング再生

```sh
mpv --volume=50 --ytdl-format="bestvideo+bestaudio" 'https://www.youtube.com/watch?v=xxxx'
```

## 2:23 AM

```sh
mpv --loop --no-video --volume=50 'https://www.youtube.com/watch?v=Yb-rLsCpBvI' # 2:23 am
```

## 3:03 PM

```sh
mpv --loop --no-video --volume=50 'https://www.youtube.com/watch?v=xxpg9_2on3I' # 3:03 pm
```

## BLACK ESCAPE

```sh
mpv --loop --no-video --volume=50 'https://www.youtube.com/watch?v=KImM8BgByvI' # black escape
```
